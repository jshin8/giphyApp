## Giphy App

Steps to run:

#### 1. Install Meteor

###### `curl https://install.meteor.com | sh`


#### 2. Clone the repo 

###### `git clone https://github.com/jshin8/giphyApp.git`


#### 3. Go to root

###### `cd giphyApp`


### 4. Install NPM

###### `meteor npm install`


#### 5. Start app

###### `meteor --settings settings.json`


### App should be running at: http://localhost:3000/
